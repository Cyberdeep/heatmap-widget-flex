Classpublic class HeatMapGradientDict
InheritanceHeatMapGradientDict Object

Contains pregenerated arrays that are used by the HeatMap's internal gradientArray, when setting the theme on the ArcGISHeatMapLayer. The fillCustomPaletteMap method may also be used to populate the "CUSTOM_TYPE" theme using an array of color values.

Method Detail
fillCustomPaletteMap()method
public static function fillCustomPaletteMap(colors:Array, alphaValue:Number = 1):void

Fills the custom palette with the colors specified in the array, the colors should be arranged left (cooler) to right (warmer). This allows you to then set the ArcGISHeatMapLayer theme to HeatMapGradientDict.CUSTOM_TYPE. 

     var customColors:Array = [ 0x7E81FF, 0x3EFF1C, 0xFFE919, 0xFF5E07, 0xFF0694, 0xFF8684 ];
     HeatMapGradientDict.fillCustomPaletteMap(customcolors);
     myHeatMapLayer.theme = HeatMapGradientDict.CUSTOM_TYPE;

Parameters

colors:Array — The array of colors to be used to fill in the heatmap.
 
alphaValue:Number (default = 1) — The transparency of the custom palette.
